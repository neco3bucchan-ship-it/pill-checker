## 開発環境の動作確認手順（100%完成版）

1. **依存関係のインストール**
   ```bash
   npm install
   ```
   初回セットアップ時のみ必要です。  

2. **開発サーバー起動**
   ```bash
   npm run dev
   ```

3. **ブラウザで表示確認**
   - `http://localhost:3000` を開き、以下を確認します。
     - ユーザー切替ボタンが表示され、編集・削除ができる
     - 時間帯ボタンが5種類表示され、クリックでハイライトが変わる
     - 選択した条件に応じて「用意する薬の内訳」が更新される
     - 薬登録フォームで画像アップロードができる
     - 登録済み薬一覧で編集・削除ができる
     - カメラ起動→プレビューが表示される（許可が必要）
     - 「カメラで撮影して自動判定」ボタンで画像認識が実行される
     - 判定履歴が保存・表示される
   - iPhone 実機でPWAとしてインストールできることを確認

4. **Lint チェック**
   ```bash
   npm run lint
   ```
   エラーが出なければ現状のコード品質は Next.js の推奨規約を満たしています。  

5. **ビルド確認**
   ```bash
   npm run build
   ```
   ビルドエラーがないことを確認

以上の項目が通れば、100% 完成版の動作確認は完了です。
